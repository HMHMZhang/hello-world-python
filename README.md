# hello-world-python

我的第一个Python项目 🐍

这是一个简单的Python Hello World程序，展示基本的Python语法和功能。

## 🚀 功能特性

- ✅ 基本的Hello World输出
- ✅ 系统信息显示
- ✅ 交互式用户输入
- ✅ 个性化问候
- ✅ 跨平台兼容

## 📋 项目结构

```
hello-world-python/
├── hello_world.py    # 主程序
├── README.md         # 项目说明
├── requirements.txt  # 依赖列表
└── .gitignore       # Git忽略文件
```

## 🎯 使用方法

### 直接运行
```bash
python hello_world.py
```

### 交互模式
程序会询问你的名字，并提供个性化问候。

## 🛠️ 系统要求

- Python 3.6+
- 支持Windows、macOS、Linux

## 📚 学习要点

- Python基础语法
- 函数定义和使用
- 模块导入
- 用户输入处理
- 字符串操作
- 系统信息获取
